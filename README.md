# Trees Detection and Segmentation using MASK-RCNN

This repo provided supplementary material for our paper Coconut Trees Detection and Segmentation in Aerial Imagery using Mask R-CNN - published in IET Computer Vision.


## Documentation

(https://ietresearch.onlinelibrary.wiley.com/doi/full/10.1049/cvi2.12028)



## Related

The repository will be updated and necessary code will be provided to the user for their tasks on tree detection. The work in this repo borrows the Mask R-CNN code from https://github.com/matterport/Mask_RCNN
